# 1. Purpose

This document provides forward-looking architectural guidance for future biological reasoning expansion within RDGP.

It is intentionally:

- strategic
- architectural
- future-aware
- semantically focused

rather than:

- implementation-prescriptive
- score-focused
- algorithmically rigid
- roadmap-binding

This document exists to help future RDGP development preserve:

- semantic extensibility
- biological realism
- explainability
- uncertainty preservation
- compositional reasoning capability
- architectural coherence

as RDGP evolves beyond v1 stabilization.

---

# 2. Core Architectural Principle

RDGP should evolve toward:

```text
a compositional semantic translational reasoning architecture
```

rather than:

- a monolithic ranking engine
- a hard-coded scoring ruleset
- a static inheritance filter
- an opaque prioritization system
- a biologically flattened evidence aggregator

Future development should preserve:

- decomposed evidence structures
- contextual biological semantics
- modular reasoning layers
- uncertainty-aware interpretation
- explainability-friendly state retention
- provenance-aware evidence composition

---

# 3. Current RDGP Biological Scope

RDGP v1 currently emphasizes:

- deterministic ranking
- transparent score decomposition
- semantic-state preservation
- confidence-aware reasoning
- inheritance-aware compatibility semantics
- phenotype-scoped prior integration
- validation-first architecture

RDGP v1 intentionally does NOT yet deeply reason about:

- biological mechanism
- developmental timing
- tissue vulnerability
- transcript-aware consequence interpretation
- dosage sensitivity
- oligogenic interaction
- temporal disease progression
- pathway-level mechanistic convergence

These are future expansion areas.

---

# 4. Important Architectural Warning

Future RDGP development should avoid allowing biological complexity to collapse into:

```text
one opaque scalar score
```

Biological interpretive dimensions should remain:

- semantically distinct
- inspectable
- explainable
- uncertainty-aware
- provenance-preserving
- validation-compatible

Whenever feasible:

```text
biological dimensions should compose
rather than collapse
```

---

# 5. Recommended Future Biological Reasoning Layers

The following future reasoning layers are considered biologically important future RDGP expansion areas.

These are intentionally NOT implementation requirements for v1.

---

# 6. Mechanistic Reasoning

## Importance

Mechanistic reasoning is likely the highest-priority future biological expansion layer.

Current RDGP understands:

- inheritance compatibility
- prioritization strength
- phenotype priors

but does NOT yet deeply reason about:

```text
what molecular mechanism plausibly explains disease behavior
```

This is a major future frontier.

### Recommended Future Mechanistic Placeholder Fields

Future RDGP mechanistic reasoning may preserve semantically decomposed mechanism-aware interpretation fields such as:

| Field | Purpose |
|---|---|
| `mechanism_context` | broad mechanistic interpretation context |
| `mechanism_support` | degree of mechanistic compatibility/support |
| `mechanism_conflict` | mechanistic contradiction or incompatibility |
| `mechanism_explanation` | explainable mechanistic interpretation summary |

These fields are intentionally future-facing and are not required for RDGP v1 implementation.

Future mechanistic reasoning should preserve distinction between:

- mechanistic plausibility
- inheritance compatibility
- confidence
- uncertainty
- prioritization strength

Mechanistic reasoning must not silently collapse into:

- hidden score suppression
- opaque pathogenicity assertions
- unexplained biological plausibility penalties
- deterministic causal claims

Future mechanism-aware reasoning should remain:

- explainable
- provenance-aware
- biologically contextual
- semantically reconstructable
- uncertainty-preserving
- modular

---

- loss-of-function semantics

# 13. Recommended Future Expansion Order

Current recommended future expansion order:

| Priority | Future Layer |
|---|---|
| 1 | mechanistic reasoning |
| 2 | phenotype reasoning |
| 3 | functional consequence reasoning |
| 4 | dosage/constraint reasoning |
| 5 | tissue/developmental context |
| 6 | oligogenic reasoning |
| 7 | temporal reasoning |

Inheritance-aware reasoning remains the correct current next-step frontier.

Mechanistic reasoning is likely the next major biological depth-expansion layer after inheritance stabilization.

---

# 14. Implementation Philosophy

Future biological expansion should proceed through:

```text
semantic stabilization
    before
algorithmic expansion
```

and:

```text
explainability preservation
    before
biological compression
```

Whenever possible:

- preserve uncertainty
- preserve provenance
- preserve decomposition
- preserve contextual semantics
- preserve reconstructability

Future RDGP should preferentially evolve through:

```text
layered compositional reasoning
```

rather than:

```text
monolithic biological scoring
```

# 16. Summary

RDGP is evolving toward:

```text
a biologically literate translational reasoning framework
```

rather than:

```text
a static ranked-gene generator
```

Future-safe RDGP architecture should therefore prioritize:

- semantic extensibility
- uncertainty preservation
- compositional reasoning
- biological contextualization
- explainability
- provenance-aware evidence layering

over:

- short-term implementation compression
- opaque scalar aggregation
- biologically flattened scoring logic
- irreversible evidence collapse

The long-term goal is not merely:

```text
better ranking
```

but:

```text
more biologically faithful reasoning
```

while preserving deterministic, explainable, validation-first computational architecture.
